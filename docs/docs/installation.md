# Installation Guide

## Prerequisites
- Node.js 
- npm

## Setup Instructions

1. Clone the repository
```bash
git clone https://github.com/JakubKonkol/movie-theater-api
cd movie-theater-api
```

2. Install dependencies
```bash
npm install
```

4. Start the server
```bash
npm run start
```

The API will be available at `http://localhost:8080`
